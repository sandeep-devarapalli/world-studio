Placement system (floating mode)
Panels float over a full-bleed 1920×1080 viewport at fixed anchor slots: .ws-top-left (wordmark + tool rail) · .ws-top-center (mode switcher; second row at top:96 for render pills) · .ws-top-right / .ws-right-col (inspectors, 408px) · .ws-left (primary list panel, 396px at 96,122) · .ws-bottom-center (timeline) · .ws-bottom-right (mode card / PiP) · .ws-statusbar (full-width). Inset = 28px (docked: 10px). Docked mode (`.docked` on .ws-root) removes blur/shadows, squares radii to 9px, solid panel fills. Compact density (`.dense`) drops paddings ~30% and font sizes ~10%.
