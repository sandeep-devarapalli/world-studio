Rules
Accent is reserved for: active/selected state, record indicators, playheads, agent overlays, primary actions. Never use accent for static decoration. Deleted/ghosted geometry renders red at 10% alpha. State colors derive from --acc via color-mix (13% fill / 55% border for selected rows).
