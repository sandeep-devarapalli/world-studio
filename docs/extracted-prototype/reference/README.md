# reference/ — visual ground truth

One screenshot per mode, captured from `World Studio App.html` at default tweaks
(floating panels, regular density, ember accent). Use these to sanity-check layout when
building the desktop app — panel anchors, spacing, accent usage.

Capture caveats: the DOM-based capturer approximates backdrop blur, so floating panels
look slightly flatter/lighter here than in the running prototype; the wordmark panel may
show a white artifact. The prototype in a browser is the final authority — these images
are for quick orientation, not pixel-diffing glass effects.
