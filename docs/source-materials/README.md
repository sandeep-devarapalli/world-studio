# Source Materials

This folder archives the materials used to bootstrap World Studio:

- `World Studio.zip`: original design prototype, design system, reference screenshots, and
  `loft_04` smoke fixture.
- `World studio development.docx`: product and architecture plan.
- `world-studio-development.txt`: text extraction of the Word plan for review.

The extracted prototype copy lives in `docs/extracted-prototype/`. The production app does
not load prototype JSX/HTML directly.

The public Budo compatibility contract is distilled into `docs/data-contracts.md` and
`docs/architecture.md`. Raw internal Budo evidence notes are intentionally kept out of the
public Apache repo because they contain local development paths and proof-only references.
