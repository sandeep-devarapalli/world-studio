# Source Materials Summary

The World Studio prototype contains the production design direction:

- `codex.md`: file map, tokens, six modes, renderer contract, desktop notes.
- `design.md`: rationale for full-bleed world-first HUD, warm-charcoal palette, typography,
  mode grammar, and copy voice.
- `ws-styles.css`: canonical CSS tokens and component classes.
- `ws-*.jsx`: prototype primitives, shell, modes, and edit workbench.
- `ws-render.js`: procedural stand-in; its API shape is the renderer contract to replace.
- `data/loft_04`: smoke fixture used by the production scaffold.

The production app copies the fixture into `apps/web/public/fixtures/loft_04` and keeps
the full prototype/archive in docs for reference.
